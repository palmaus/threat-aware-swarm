# Environment Schema

Этот документ фиксирует контракт наблюдений и минимальные правила совместимости моделей.

## Текущий стандарт

| Поле | Размер |
| --- | ---: |
| `to_target` | 2 |
| `vel` | 2 |
| `walls` | 4 |
| `last_action` | 2 |
| `measured_accel` | 2 |
| `energy_level` | 1 |
| `grid` | `41 x 41 = 1681` |
| **Итого** | **1694** |

Формат версии:
```text
obs@<dim>:v<k>
```

Текущий контракт:
```text
obs@1694:v5
```

## Семантика grid
- стены: `1.0`;
- угрозы: примерно `0.7-0.8`;
- другие агенты: `0.3`;
- self: `0.1`.

## Правила совместимости
- поддерживается только `obs@1694:v5`;
- legacy `obs@129` и `obs@449` удалены из рабочего контура;
- если меняется размерность, увеличивай `dim`;
- если меняется порядок или смысл полей, увеличивай `v<k>`.

## Где хранится версия
- основной источник: `<model>.meta.json`;
- fallback: `runs/<run_id>/meta/run.json`.

## Что считается breaking change
- перестановка полей без увеличения версии;
- изменение размера grid без обновления `obs_dim`;
- тихая адаптация legacy-моделей в train/eval runtime.
